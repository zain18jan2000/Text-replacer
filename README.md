# Text-replacer
A c program to find and replace a word in a File by another given word <br/>

A text replacing program can be helpful when we need to replace a word such as name by of a person by another word or name at more than one locations in a file.
<br/>
This program can replace a word at all the locations in a file of extension '.txt' by another one. The program can also be used for other files such as json and html with some modification in the program. To use this program for html, dat or json file following changes are needed in the source code 'main.c'.<br/>
1- In line 17,20,73 and 78 replace the '.txt' by the extension of the file you want to edit.<br/>
2- recompile the source code to obtain .exe file.
